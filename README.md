# Round Robin Scheduler

React app for Round Robin CPU scheduling visualization.

**Link**: [https://grace-anna.github.io/Round-Robin-Scheduling/](https://grace-anna.github.io/Round-Robin-Scheduling/)

## Features
- Add/delete processes
- Animated Gantt chart  
- Live ready queue
- Full metrics table

## Run
```bash
npx create-react-app rr-project
# Replace src/App.js
npm start
```

## Usage
1. Set time quantum
2. Add processes  
3. Click Calculate
4. Watch animation
5. Review results table
